# Study Documents

This folder contains study materials and documentation for the AZ-305 certification exam.

## Organization

Documents are organized by the main exam objective areas:

### 1. Identity, Governance, and Monitoring
- `identity-and-access.md` - Azure AD, RBAC, Managed Identities
- `governance.md` - Azure Policy, Blueprints, Management Groups
- `monitoring.md` - Azure Monitor, Log Analytics, Application Insights

### 2. Data Storage Solutions
- `relational-storage.md` - Azure SQL, Cosmos DB (SQL API)
- `non-relational-storage.md` - Cosmos DB, Table Storage, Blob Storage
- `data-integration.md` - Azure Data Factory, Event Hub, Event Grid

### 3. Business Continuity
- `backup-recovery.md` - Azure Backup, Site Recovery
- `high-availability.md` - Availability Zones, Availability Sets, Load Balancing

### 4. Infrastructure Solutions
- `compute.md` - VMs, App Services, Containers, Kubernetes
- `networking.md` - VNets, VPN, ExpressRoute, Application Gateway
- `application-architecture.md` - Microservices, Serverless, API Management
- `migrations.md` - Azure Migrate, Database Migration Service

## Study Tips

1. Read each document thoroughly
2. Create your own notes and summaries
3. Cross-reference with official Microsoft documentation
4. Practice drawing architecture diagrams
5. Complete hands-on labs to reinforce concepts
